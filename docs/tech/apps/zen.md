# Zen (ブラウザ)
FirefoxベースのコミュニティOSSブラウザ。

今のところ私はZenは初心者なので多くは語れないが、垂直タブ・ワークスペース・上にバーがないところ等のZenらしさを結構気に入っている。

## Firefox Sync
私はモバイル (Android, iPad) ではFirefoxを使うので、ZenにもFirefox Syncがあって助かる。Settingsの「Account and sync」からできる。

## 拡張機能
Firefoxのものがそのまま使える。Bitwarden, Foxy Proxy, Vimium等のお気に入りがそのまま使えて嬉しいね。

Zen専用の拡張機能的なのは[Zen Mods](https://zen-browser.app/mods/)にある。CSSを当ててUIを良くしようとしているものが多そうだが今のところ自分は何も取り入れていない。

## テーマ
Firefox用のテーマだとZenのものには対応しきれないので注意。いつものCatpuccinは[ここ](../ui/catppuccin.md#zen)から。
